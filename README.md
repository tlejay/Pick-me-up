# PICK-ME-UP

AI-Powered Coffee Pick-up Platform

## Overview

PICK-ME-UP is an intelligent coffee ordering and pickup platform that leverages AI agents to streamline the entire customer journey from browsing to pickup.

## Project Structure

- **docs/** - All project documentation
- **backend/** - Node.js backend services and AI agents
- **mobile/** - Flutter mobile application
- **web/** - Web application (React/Vue)
- **infrastructure/** - DevOps configurations
- **shared/** - Shared types and constants
- **scripts/** - Utility scripts
